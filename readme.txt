== Description ==

Amazing Portfolio is an easy to use responsive and filterable Portfolio-Grid Plugin for WordPress, offering a wide range of customization options.


>[Live Demo ](https://themebon.com/amazing-portfolio/ "Amazing Portfolio")


<strong>Features</strong>
<ul>
    <li>6 Columns Grid.</li>
    <li>Image, Gallery, Video portfolio type</li>
    <li>100% Responsive</li>
    <li>Super easy Installation.</li>
    <li>Unlimited portfolios.</li>
    <li>Portfolio filter 3 styles.</li>
    <li>No column space.</li>
    <li>Portfolio home page gallery.</li>
    <li>Title on hover.</li>
    <li>Pagination support.</li>
    <li>Category support.</li>
    <li>Related items.</li>
    <li>Number of post to show.</li>
    <li>Show item by old or new.</li>
    <li>Tons of customization options.</li>
    <li>Powerfull Shortcodes Generator.</li>
    <li>SEO friendly.</li>
    <li>Easy user interface.</li>
    <li>Works with any theme.</li>
    <li>All Major browser supported.</li>

</ul>



== Installation ==

1. Upload “amazing-portfolio” plugin to the /wp-content/plugins/ directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Add a few Portfolio entries in Amazing Portfolio -> Add New”
4. Create New Page and there should be a button "Add Portfolio Shortcode"
5. Click that button and generate your shortcode and publish the page to see output
